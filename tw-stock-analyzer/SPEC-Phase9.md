# Phase 9: 後台管理系統

## 概述
為管理員提供後台系統，管理會員、訂單、內容等功能。

---

## 功能清單

### 1. 會員管理
- 會員列表（搜尋/篩選/分頁）
- 會員詳情（資料編輯、等級變更）
- 會員啟用/停用
- 會員刪除

### 2. 訂單管理
- 訂單列表
- 訂單詳情
- 訂單狀態變更
- 營收統計

### 3. 訂閱管理
- 訂閱列表
- 訂閱展延
- 取消訂閱
- 訂閱統計

### 4. 價格提醒管理
- 提醒列表
- 啟用/停用
- 清理過期提醒

### 5. AI 使用統計
- 每日使用次數
- Token 消耗統計
- 快取命中率

### 6. 儀表板
- 總會員數
- 活躍會員數
- 本月營收
- 訂單趨勢圖

---

## 技術架構

### 後端
- 新增 `/api/admin` 路由
- 管理員權限檢查
- 統計 API

### 前端
- 新增 `/admin` 頁面
- 管理員儀表板
- 資料表格（排序/篩選/分頁）
- 圖表顯示

---

## API 設計

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | /api/admin/dashboard | 儀表板統計 |
| GET | /api/admin/users | 會員列表 |
| GET | /api/admin/users/:id | 會員詳情 |
| PUT | /api/admin/users/:id | 更新會員 |
| DELETE | /api/admin/users/:id | 刪除會員 |
| GET | /api/admin/orders | 訂單列表 |
| GET | /api/admin/orders/:id | 訂單詳情 |
| GET | /api/admin/subscriptions | 訂閱列表 |
| GET | /api/admin/alerts | 提醒列表 |
| GET | /api/admin/ai-stats | AI 使用統計 |

---

## 會員權限

| 等級 | 權限 |
|------|------|
| user | 一般會員 |
| admin | 後台管理員 |
| superadmin | 超級管理員 |
